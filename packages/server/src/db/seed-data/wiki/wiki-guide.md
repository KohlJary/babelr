# Using the Wiki

The wiki is where durable knowledge lives — the things that shouldn't scroll away in chat. Every server has its own wiki with nested pages, revision history, comments, and full-text search.

## Creating Pages

Click **"+ New page"** in the wiki sidebar. Give it a title (the URL slug is generated automatically), write your content in markdown, and save.

### Nested Pages

Pages can be children of other pages. Set a parent when creating to build a hierarchy:

```
Getting Started
├── Channels & Messaging
├── Using the Wiki
├── Calendar & Events
└── Server Files
```

The sidebar renders this as a collapsible tree. Breadcrumbs above the page content show the ancestry trail.

## Editing

Click **"Edit"** to modify a page. The editor supports:

- Full markdown with live preview (toggle Write/Preview)
- File attachments via the **"Attach file"** button, drag-and-drop, or clipboard paste
- Images insert as `[[img:slug]]` embeds that render inline with lightbox
- Other files insert as `[[file:slug]]` embeds with download cards

## Revision History

Every edit is recorded. Click **"History"** to see all revisions with who changed what and when. Click any revision to view the old content, and **"Restore this version"** to roll back (this creates a new revision — history is never deleted).

## Comments

Each page has a discussion thread below the content. Comments use the full message pipeline — reactions, threads, typing indicators, and translation all work.

## Tags

Add tags to pages for organization. The sidebar shows a tag filter — click a tag to narrow the page list.

## Wiki Links

Type `[[wiki:page-slug]]` in any wiki page (or chat message) to create a cross-reference. The **Backlinks** section at the bottom of each page shows everything that links to it — both wiki pages and chat messages.

## Table of Contents

Pages with 2 or more headings automatically get a table of contents above the content. Click any entry to scroll to that section.
