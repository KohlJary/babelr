# Getting Started

This guide walks you through the core surfaces of Babelr. Each section below has its own detailed page.

## The Sidebar

The left sidebar shows your **servers** (the communities you belong to). Click a server to see its channels. The sidebar also has entries for:

- **Calendar** — server events with RSVP
- **Wiki** — the page you're reading right now
- **Files** — shared file library with comments

## Your First Steps

1. **Join a server** or create your own from the "+" button
2. **Set your preferred language** in Settings — this controls what language content translates into for you
3. **Start chatting** in #general — messages from other languages translate automatically
4. **Explore the wiki** — you're already here! Browse the sidebar tree on the left
